weblinda by weblinda

WebLinDa is an inter-linear gloss database for working with collected linguistics data. It has modes for processing individual morphemes all the way to long texts.

Documentation

RDoc, or documentation generated directly from the source, is available. WebLinDa is a pretty standard Rails app, so the official documentation is also a good resource.
